# ⭐ Interactive Rating Component

An interactive rating component built as part of a **Frontend Mentor** challenge. Users can select a rating from **1–5** and submit their feedback to view a personalized thank-you message.

## 🚀 Features

* Select a rating between **1 and 5**
* Highlights the selected rating
* Displays a thank-you state after submission
* Responsive design for mobile and desktop
* Smooth hover and button animations
* Accessible keyboard focus styles

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)
* Flexbox
* CSS Custom Properties (Variables)
