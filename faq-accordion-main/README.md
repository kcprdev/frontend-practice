# FAQ Accordion

A responsive FAQ Accordion built using **HTML, CSS, and JavaScript**. Users can expand and collapse FAQ items with smooth interactions, and only one answer is displayed at a time.

## ⚡ Challenges

One challenge was ensuring that only one FAQ item remained open at any time. Initially, multiple sections could stay expanded. I solved this by closing all other accordion items before opening the selected one and updating the icons accordingly.
